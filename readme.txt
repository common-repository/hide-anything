== Description ==
If you don't know coding or want to hide any unnecessary element on your site then this plugin is for you.
You can hide any element on any page by visually selection with a simple click without any coding.

The user won't see the hidden elements when they will browse the site until you make it visible again.

**Frontend Hiding** - Hide any element on the website frontend.

**Admin Dashboard Hiding** - Hide any element on the admin dashboard.

**Visual Element Selection** - You can select any element visually by clicking on it to hide.

**No Coding** - You don't need to know any coding to hide any element.

**Restore the Hidden Element** - You can restore the hidden element by clicking on it.

**Multiple Display (Desktop, Tablet, Mobile) Options** - You can choose the display options (Desktop, Tablet, Mobile) where the element will be hidden.


== HOW TO USE? ==

1. Install and activate the plugin.

2. To hide any element in any page on your site frontend, Go to the page where you want to hide any element as a logged-in user with admin privileges.

3. You will see a "Hide Anything" button in the top admin bar. Click the button to activate the element selection.

4. You can see a blue overlay area of the element while the mouse moving on any element, you have to click the element.

5. Now you can see that a display choose option appeared under the "Hide Anything" button.

6. Then click the save button.

7. To restore the element, you have to click the "Hide Anything" button to activate the element selection.

8. You can see that the hidden element is overlayed with a red color. You have to click the element.

9. Now, you will get an option (Make Visible) under the "Hide Anything" button, you have to check the Make Visible option and save the settings.
